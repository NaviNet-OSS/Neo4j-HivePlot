draw(args)

Draw the hive plot based on the provided arguments.

Parameters:
Name Type Description
args object

The arguments that describe the plot to be drawn.

Properties
Name Type Argument Description
graph Neo4j.Graph.Graph

The object graph containing the data to be plotted.
nodeDomain string[]

The list of node domain keys to be included in the plot.
relationshipDomain string[]

The list of relationship domain keys to be included in the plot.
containerSelector string

The DOM selector for the DOM element that is to contain the plot. Note that all HTML content in this container will be deleted.
getNodeName HivePlot.getNodeNameCallback <optional>

The callback function to get the display name of each node. If omitted, the node domain key and index number will be used.
getNodeLabel HivePlot.getNodeLabelCallback <optional>

The callback function to get the display label of each node. If omitted, the node domain key will be used.
getRelationshipType HivePlot.getRelationshipType <optional>

The callback function to get the display name of the relationship type. If omitted, the type indicated in the graph will be used.
